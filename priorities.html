Our Plan for Biddeford

These are Sam’s core priorities: practical maintenance, smart growth, and accountable results. Each priority outlines goals and steps we can measure.

  • Priority #1 — Fix What’s Broken
  • Priority #2 — Grow Smart
  • Priority #3 — Own the Results
▶ Priority #1 — Fix What’s Broken

Subtitle (optional): Routine maintenance first, predictable costs, credible timelines.

Biddeford’s growth hasn’t been felt equally, and deferred infrastructure risks more tax hikes. Finish mandated sewer and street work wisely, stand up a maintenance line in the budget, and publish quarterly snapshots (scope, spend‑to‑date, next milestones).

Stop the bleeding: pause non‑essential projects; maintenance lockbox; public backlog dashboard.
▶ Priority #2 — Grow Smart

Subtitle (optional): Enable housing where it fits; align code to outcomes.

Slow down the mega‑projects. Build the middle — 2–4 unit infill and small mixed‑use near downtown — so owner equity stays local. Apply clear standards and a measured pace: design fit, infrastructure capacity, and fair‑share contributions. Add anti‑displacement safeguards so no one is priced out.

▶ Priority #3 — Own the Results

Subtitle (optional): Transparency, measurable goals, public reporting.

Say what we’ll do. Do it. Report back. Project scorecards for major work, plain‑English explanations before votes, and after‑action reviews when we miss. If we get it wrong, we own it and fix it.
